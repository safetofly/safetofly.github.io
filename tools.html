STLCG (Signal Temporal Logic Computation Graph)

STLCG is a powerful Python toolbox that represents Signal Temporal Logic (STL) robustness formulas as computation graphs. By leveraging PyTorch, it enables back-propagation through logic specifications, allowing for a seamless integration of formal requirements into gradient-based optimization and machine learning workflows.
